 # Descripcion general del sistema: 
 Nuestro programa tiene como propósito administrar reservas de entradas para distintas funciones de cine, permite que el usuario elija que película ver,  el usuario puede visualizar estadísticas del dia, qué película tuvo más reservas, controlando la capacidad de cada sala. Y luego salir del sistema si no desea reservar.

Funcionalidades principales
 - Realizar reservas de entradas.
 - Seleccionar película y horario.
 - Registrar las reservas en un archivo de texto.
 - Controlar la capacidad disponible de las salas.
 - Consultar estadísticas básicas de ventas.
 - Salir de sistema.
